# Финальный проект 7 спринта. Автоматизация тестирования API Яндекс Самокат.

В этом проекте написаны автотесты для API сервиса "Яндекс Самокат" с использованием Python, pytest и библиотеки requests.

# Что протестировано:
1. Создание курьера
2. Логин курьера
3. Создание заказа (с параметризацией цветов)
4. Получение списка заказов
*Дополнительно:* удаление курьера, принятие заказа, получение заказа по трек-номеру.

# Запуск тестов:
Установить зависимости: `pip install requests pytest allure-pytest`
Запустить тесты: `pytest --alluredir=allure_results`
Посмотреть отчет: `allure serve allure_results`
